腦痙攣男童抱籃球大笑

今年10歲、患有腦痙攣的黃景寧，性格開朗活潑，令記者留下深刻的印象。景寧的聖誕節願望是想要一個籃球，母親黃太表示，景寧雖然患有痙攣，但喜愛玩球，「儲球」更是其「嗜好」，他閒時愛坐在椅上，與弟弟玩擲球。現時家中已收集各類球，景寧下一個目標是「壁球」。當記者蹲下問景寧是否喜歡其聖誕禮物時，他整個人即在椅上彈起，大叫：「好開心！」
同樣患有痙攣的慧詩樣子長得佷標致，惹人喜愛。慧詩母親鄺太表示，女兒今年10歲，性格樂天，容易滿足，別人讚她「靚女」已令她樂上半天。慧詩喜愛閱讀，她的聖誕願望是擁有一本米奇系列的偵探故事書。鄺太稱，她經常到圖書館借書返家與女兒一同閱讀，她大讚女兒記性很好，聽完故事後每每記得內容，「佢特別鍾意聽米奇嘅古仔，我今晚返到去都會讀畀佢聽，等佢開心吓。」
